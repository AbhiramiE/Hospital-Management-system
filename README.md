# 🏥 Hospital Management System

A full-stack **Hospital Management System** designed to streamline hospital operations such as patient management, appointment scheduling, and administrative control. The project is structured with separate frontend, backend, and database modules for scalability and maintainability.

---

## 🚀 Features

- 🔐 User Authentication & Role-Based Access Control  
- 👩‍⚕️ Patient Registration & Record Management  
- 📅 Appointment Scheduling System  
- 🏥 Doctor & Staff Management  
- 🔄 Full CRUD Operations (Create, Read, Update, Delete)  
- 🌐 RESTful API Architecture  
- 🗄️ Database Integration for Persistent Data Storage  

---

## 🧰 Technology Stack

| Layer      | Technology Used |
|------------|----------------|
| Frontend   | React |
| Backend    | Flask|
| Database   | postgres sql |
